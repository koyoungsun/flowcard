# 📱 FlowCard — 하루를 카드로 여는 데일리 런처

**FlowCard**는 알람처럼 하루를 시작하며, 카드 형태로 날씨·뉴스·링크 등을 한눈에 확인하고 실행할 수 있는 **개인화 데일리 뷰어**입니다.

Vue 3 + TypeScript + Vite 기반으로 개발되며, 심플하고 빠른 사용성을 추구합니다.

---

## 🔥 주요 기능 (1차 버전)

- [x] 카드형 / 리스트형 보기 모드 전환
- [x] 카드 추가 / 편집 / 삭제
- [x] 로컬 스토리지 기반 카드 저장
- [x] 기본 보기 모드 설정 (카드 or 리스트)

---

- [ ] 자주 가는 앱/사이트 런처 카드
